## 关于卡组导入
- 你可以通过创建一个文本文件（deck.txt，且与wsfStS.exe放置于同一目录）来导入自己的卡组，文件格式如下：
  - 第一行是一个数字N，表示卡组中有多少张牌
  - 之后N行，每行表示一张牌，仅包含牌名（无前后空格，并使用下划线代替空格）
  - 最后空一行表示卡组结束
- 程序不会检查卡组是否有效，请务必按照格式正确填写。
